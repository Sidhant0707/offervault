# Security Policy

## Supported Versions
We currently support and provide security updates for the latest version of OfferVault (v1.x).

## Security Measures
- **Row-Level Security (RLS):** All database transactions are governed by Supabase RLS policies.
- **Data Hashing:** Sensitive identifiers are hashed to maintain student anonymity.
- **Input Validation:** Strict server-side regex validation for .edu/.ac.in domains.
